## 核心能力总览

| 维度 | 能力 | 说明 |
|------|------|------|
| **1-5-10 战略** | 发现 / 定位 / 恢复 | 1 分钟发现、5 分钟定位、10 分钟恢复 |
| **4 层架构** | 采集层 / 数据层 / 能力层 / 应用层 | 分层解耦、模块自治 |
| **11 个模块** | 数据融合 → 知识进化 | 完整运维可观测闭环 |
| **6 类 Agent** | Orchestrator / Monitor / Analyst / Investigator / Executor / Notifier | 多 Agent 协作 |
